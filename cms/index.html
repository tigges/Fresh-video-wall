What to edit where

CMS App (WordPress)

  • Hero tagline and marketing copy
  • Section titles and CTA labels/URLs
  • Social media links and on/off status
  • Contact page intro text and form action target

Main App (Code Repo)

  • Layout, styling, spacing, and button component design
  • Media feed rendering and playback behavior
  • Automation workflows and deployment logic
  • Page structure and new feature modules

Rule of thumb: if it is text/link content, edit in CMS app; if it changes behavior or design, edit in the main app codebase.
